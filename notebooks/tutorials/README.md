# Tutorial Notebooks

These notebooks are API-first walkthroughs. They explain how to use Cave,
generate artifacts under `out/tutorials/`, and should import package/report
logic instead of owning experiment implementations.

- `01_intro_to_cave_subjective_trajectory.ipynb`: builds an external sequence,
  configures a subject, runs `CaveProducer`, inspects `Episode` state, renders
  the walkthrough views, relates authored qualities to resolved effects, lightly
  derives internal expectation objects from the produced episode, and compares
  attention configurations.
- `02_comparing_experiences.ipynb`: compares same-world/different-subject and
  same-subject/different-experience runs, explains memory/effect and embedding
  readouts, and renders distance dashboards, topology atlas diagnostics, and
  population trajectory geometry.
- `03_pressures_cavenet_evolved_subjects.ipynb`: walks through CaveNet
  pressure-shaped gains, role recovery evidence, evolved recurrent exposure
  controls, and cross-substrate episode comparisons.
